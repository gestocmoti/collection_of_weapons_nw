サーモンランブキコレクション
====

サーモンランのランダム回で引いたブキを数えたい人用のツール。

## 説明

- ブキの画像をタップすると、画像の右上に数字が出ます。
- たくさんタップすると、右上の数字が増えていきます。
- アンドゥ/リドゥができます。
- 表示するクマサン印のブキを選べます。
- 右上の数字を隠せます。
- 1個のブキを2回以上カウントする必要がない、という場合には、「カウント済みのブキをタップしたときの動作」を「カウントを減らす」にして「カウントを隠す」のがおすすめです。
